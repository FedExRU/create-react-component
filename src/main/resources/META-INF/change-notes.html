Features

  • Create React component with specified name;
  • Ability to create React component with necessary code dependent on selected options;
  • Ability to create types file for component (types.ts);
  • Ability to create constants file for component (constants.ts);
  • Ability to create utility file for component (utils.ts);
  • Ability to specify memo component;
  • Ability to specify memo functional component;
  • Ability to create additional directories for module component.
